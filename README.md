# BF528 Project 2: RNA-seq Pipeline and Differential Expression Analysis

## 📌 Project Overview
This repository contains a complete, automated end-to-end RNA-seq analysis pipeline and downstream statistical analysis developed for BU BF528 (Genomic Data Analysis). The pipeline is designed to process raw sequencing reads, align them to the human reference genome, quantify gene expression, and perform differential expression analysis. 

The ultimate goal of this project is to successfully process raw data from a published study and reproduce **Figure 3C** and **Figure 3F** from the original *Nature Communications* paper: *"The type 1 diabetes gene TYK2 regulates β-cell development and its responses to interferon-α"* (GSE190725).

## 🧬 Biological Context
- **Condition:** Human induced pluripotent stem cells (hiPSCs) differentiated into pancreatic endocrine progenitors (Stage 5 / S5).
- **Experimental Group:** TYK2 Knockout (KO)
- **Control Group:** Wild Type (WT)
- **Objective:** Determine the transcriptomic impact of TYK2 ablation during pancreatic lineage differentiation.

---

## 🛠️ Pipeline Architecture
The upstream data processing is fully automated using **Nextflow (DSL2)** and isolated within **Singularity** containers. 

### 1. Data Preprocessing & Alignment (Nextflow)
- **Quality Control:** `FastQC` (v0.11.9)
- **Genome Indexing & Alignment:** `STAR` (v2.7.9a) against Ensembl GRCh38 Primary Assembly.
- **Quantification:** `VERSE` (v0.1.5) using Ensembl release 110 GTF annotation.
- **Matrix Aggregation:** Python `pandas` script to merge individual `.exon.txt` files into a unified raw count matrix.
- **QC Aggregation:** `MultiQC`

### 2. Downstream Statistical Analysis (R / Rmarkdown)
- **Filtering:** Removal of low-count/uninformative genes.
- **Normalization & QC:** `rlog` / `vst` transformation, PCA, and Sample-to-Sample distance heatmaps.
- **Differential Expression:** `DESeq2` (Wald test, TYK2 KO vs WT).
- **Functional Enrichment:** `fgsea` using MSigDB C2 Canonical Pathways.
